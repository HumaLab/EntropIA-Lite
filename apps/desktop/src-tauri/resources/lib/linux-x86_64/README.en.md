# Pdfium for Linux

[Español](README.md) | English

`libpdfium.so` is the official PDFium binary for Linux x86_64, build `chromium/7543`
(144.0.7543.0 — the same version as the Windows `../pdfium.dll`), taken from
[bblanchon/pdfium-binaries](https://github.com/bblanchon/pdfium-binaries/releases/tag/chromium%2F7543).
The license lives in `../LICENSE`.

When updating `pdfium.dll`, update this `.so` to the same `chromium/<rev>` build.

This is not a local ML runtime pack. Do not place Python payloads, model caches, ONNX, Paddle, faster-whisper, spaCy, GGUF or llama.cpp files for EntropIA Lite here.
