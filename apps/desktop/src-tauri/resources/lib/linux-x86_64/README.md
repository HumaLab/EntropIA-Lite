# Pdfium para Linux

Español | [English](README.en.md)

`libpdfium.so` es el binario oficial de PDFium para Linux x86_64, build `chromium/7543`
(144.0.7543.0 — la misma versión que `../pdfium.dll` de Windows), tomado de
[bblanchon/pdfium-binaries](https://github.com/bblanchon/pdfium-binaries/releases/tag/chromium%2F7543).
La licencia está en `../LICENSE`.

Al actualizar `pdfium.dll`, actualizar este `.so` a la misma build de `chromium/<rev>`.

No es un runtime pack local de ML. No coloques acá payloads de Python, caches de modelos, ONNX, Paddle, faster-whisper, spaCy, GGUF o llama.cpp para EntropIA Lite.
